🔧 Firebase 실시간 동기화 설정

아래 단계를 따라 설정하면 팀원 누구든 체크한 내용이 즉시 모두에게 반영됩니다.

1
Firebase 프로젝트 생성
console.firebase.google.com 접속 → 프로젝트 추가 → 이름 입력 (예: essay-progress) → Google 애널리틱스 비활성화 → 프로젝트 만들기
2
웹 앱 등록 및 설정 코드 복사
프로젝트 홈에서 </> 웹 아이콘 클릭 → 앱 닉네임 입력 → 앱 등록 → 표시되는 firebaseConfig 코드 전체 복사
3
Realtime Database 생성
왼쪽 메뉴 빌드 → Realtime Database → 데이터베이스 만들기 → 위치: asia-southeast1 → 테스트 모드로 시작
4
index.html에 설정 붙여넣기
이 파일 하단 <script> 안의 FIREBASE_CONFIG 객체를 복사한 값으로 교체 후 저장
5
GitHub에 푸시
수정된 index.html을 GitHub에 업로드하면 완료! 이후 누구든 URL을 열면 실시간으로 동일한 진도를 확인·수정할 수 있습니다.
